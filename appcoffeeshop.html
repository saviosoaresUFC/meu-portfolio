AppCoffeeShop

O AppCoffeeShop é um aplicativo móvel de comércio eletrônico focado na venda de café. Desenvolvido com React Native e TypeScript, ele oferece uma experiência de compra nativa, rápida e intuitiva.

Ver no GitHub (Código)

Funcionalidades Principais

Carrinho Inteligente

Gestão de itens no carrinho com cálculo de frete dinâmico e opções de personalização de pedidos (leite, açúcar, etc.).

Histórico de Pedidos

Visualização de pedidos anteriores, facilitando a repetição de compras favoritas com um único clique.

Programa de Fidelidade

Sistema de pontos e recompensas integrado, incentivando a fidelidade e oferecendo descontos exclusivos.

Busca de Lojas Próximas

Usa a geolocalização do usuário para encontrar a cafeteria mais próxima com o menu completo e horários de funcionamento.

Tecnologias Utilizadas

React Native TypeScript Expo Go React Navigation Redux Toolkit (Gestão de Estado) Firebase API de Pagamento (Stripe/Pix) Git / GitHub
