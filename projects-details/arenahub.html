O ArenaHub é uma plataforma moderna que transforma a gestão de arenas esportivas e a experiência dos atletas. Desenvolvido em Spring Boot + NextJs, combina escalabilidade, segurança e performance em um único sistema.

Ver no GitHub Visitar Site

Funcionalidades Principais

Agendamento Inteligente

Atletas podem agendar quadras facilmente, com visualização de horários disponíveis e gerenciamento simplificado para administradores da arena.

Pagamento PIX Integrado

Confirmação de agendamentos via pagamento PIX direto na plataforma, garantindo segurança e agilidade, com cancelamento automático de 10 minutos se não pago.

Jogos Abertos e Conexão

Recurso para atletas entrarem em "jogos abertos" com vagas remanescentes, promovendo a interação e garantindo quadras cheias.

Gestão de Clientes e Quadras

Ferramentas para arenas gerenciarem seus clientes, quadras, visualizar histórico de agendamentos e participação.

Avaliações e Feedback

Sistema de avaliação para atletas darem feedback sobre arenas, ajudando na melhoria contínua dos serviços.

Sorteio de Times

Funcionalidade para sortear jogadores em times, tornando a organização dos rachas mais justa e divertida.

Tecnologias Utilizadas

Java 17 & OOP Spring Boot 3 (Maven) PostgreSQL & Hibernate REST API & JWT Asaas API (PIX) Stripe (Assinatura) Zenvia (SMS) TypeScript Next.js / React Tailwind CSS Docker Git / GitHub Scrum / Metodologias Ágeis Flyway (Migrações)
