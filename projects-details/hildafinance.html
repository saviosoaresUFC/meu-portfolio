Hilda Finance

O Hilda Finance é um aplicativo móvel para gestão simplificada de Estoque e Vendas Diárias, desenvolvido sob medida para uma espetaria em Floriano, PI. Criado com React Native (TypeScript), garante agilidade e confiabilidade na operação do dia a dia.

Ver no GitHub

Funcionalidades Principais

Controle de Estoque

Gerenciamento de entradas e saídas de produtos. Alerta automático para itens com baixo estoque, reduzindo perdas e garantindo disponibilidade.

Registro de Vendas Diárias

Interface rápida e intuitiva para registrar as vendas do dia, ajudando no fechamento de caixa e na conciliação financeira.

Cadastro de Produtos Personalizado

Módulo para adicionar novos itens ao cardápio com variação de preços, descrições e categorias específicas para espetaria.

Relatórios e Dashboards

Visualizações simples e claras do desempenho de vendas, dos produtos mais vendidos e do histórico de estoque.

Tecnologias Utilizadas

React Native TypeScript Expo SQLite / AsyncStorage Context API / Redux Styled Components
