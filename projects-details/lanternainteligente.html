Lanterna Inteligente: Controle por Sensores

Mini-aplicativo Mobile nativo desenvolvido em Kotlin e Jetpack Compose. O projeto foca em Interação com Hardware, utilizando o Sensor de Luz para automatizar o controle da lanterna do dispositivo.

Ver no GitHub (Código)

Funcionalidades e Requisitos

Controle Automático (Sensor)

Detecta a luminosidade ambiente e liga/desliga a lanterna automaticamente se o ambiente estiver escuro.

Controle Manual e Display

Permite ligar/desligar a lanterna via botão manual e exibe o nível atual de luminosidade detectado.

Gerenciamento de Permissões

Gerenciamento de acesso à câmera/lanterna e ao sensor de luminosidade, conforme a API do Android.

Tecnologias Utilizadas

Kotlin (Natvio) Android SDK Jetpack Compose (UI) Sensores de Hardware API da Câmera (Flashlight)
