Locais Favoritos: App de Mapeamento

O Locais Favoritos é um aplicativo Mobile nativo, desenvolvido em Kotlin, focado na exploração e gerenciamento de pontos de interesse. O projeto utiliza intensivamente a API do Google Maps para fornecer uma experiência de mapeamento precisa e interativa.

Ver no GitHub (Código)

Funcionalidades Principais

Marcação Dinâmica no Mapa

Adicione novos pontos de interesse (marcadores) com um toque longo no mapa, registrando a latitude e longitude.

Lista de Locais Salvos

Visualize e gerencie todos os locais marcados em uma lista organizada, facilitando o acesso rápido.

Integração com Google Maps

Uso da API do Google Maps para converter coordenadas em endereços legíveis para o usuário.

Tecnologias Utilizadas

Kotlin (Natvio) Android SDK Google Maps API SQLite / Room Persistence Asaas API (PIX)
