HubStock

O HubStock é um gerenciador de estoque completo para restaurantes e bares. Desenvolvido com Vue 3 + Node.js (Express), permite o controle preciso de entradas, saídas e vendas, oferecendo relatórios detalhados para tomada de decisão.

Código no GitHub

Funcionalidades Principais

Controle de Estoque

Gerenciamento completo de produtos, categorias e movimentações de entrada e saída.

Ponto de Venda (PDV)

Interface para garçons e administradores registrarem vendas de forma rápida e eficiente.

Relatórios de Vendas

Visualização de lucros e vendas por período (dia, mês, ano) para análise de desempenho.

Gestão de Usuários

Controle de acesso com diferentes perfis: Garçom, Administrador e SuperAdministrador.

Tecnologias Utilizadas

Vue 3 TypeScript Pinia Node.js (Express) TypeORM SQLite JWT
