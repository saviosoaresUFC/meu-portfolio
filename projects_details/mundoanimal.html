Mundo Animal: Sistema para Petshop

O Mundo Animal é um sistema Desktop desenvolvido em JavaFX e PostgreSQL para otimizar a gestão de Petshops, garantindo o controle total sobre agendamentos, clientes, serviços e secretários. Focado em estabilidade e uma interface intuitiva.

Ver no GitHub (Código)

Funcionalidades Principais

Autenticação e Permissões

Login seguro para diferentes níveis de acesso: Administradores (gestão total) e Secretários (gestão de agenda e clientes).

Agendamento Dinâmico

Gerenciamento de agendamentos de serviços com exibição dinâmica dos próximos atendimentos diretamente na tela inicial.

Gestão de Clientes e Pets

Cadastro e controle eficiente de clientes e seus respectivos animais (pets), mantendo um histórico detalhado.

Controle de Serviços e Relatórios

Controle de todos os serviços realizados e relatórios de operações para tomada de decisão.

Interface Intuitiva (JavaFX)

Interface de usuário moderna e intuitiva, desenvolvida em JavaFX, com foco em usabilidade desktop.

Tecnologias Utilizadas

Java 23 JavaFX PostgreSQL Maven Git / GitHub SceneBuilder CSS
